# Origami Flight Demo ✈️

Three.js で作成された、美しく心地よい紙飛行機の飛行シミュレータープロトタイプです。
『Paper Sky』の心地よい世界観に憧れて、Three.jsで手書き風のローポリのフライトゲームのファンメイド・モックアップを作ってみました。
砂漠、森、雲海を抜けて、静かな夜の宇宙までシームレスに旅ができます。

## 🎮 プレイ方法
以下のURLからブラウザ上で直接プレイできます：
👉 [https://harukaappscreate.github.io/origami-flight-demo/](https://harukaappscreate.github.io/origami-flight-demo/)

### 操作方法
* **PC（キーボード）**
  * `W` / `↑` : 上昇（ピッチアップ）
  * `S` / `↓` : 下降（ピッチダウン）
  * `A` / `←` : 左旋回
  * `D` / `→` : 右旋回
  * `Space` : ブースト（加速）
* **スマートフォン / タブレット（タッチ操作）**
  * 画面左下の**バーチャルジョイスティック**をドラッグして操縦
  * 画面右下の**「ブースト」ボタン**を押し続けて加速

## 🛠️ 技術スタック
* **WebGL ライブラリ**: [Three.js](https://threejs.org/) (Vanilla ES modules)
* **スタイル & UI**: Vanilla CSS / HTML5 (ペーパーテイストな手描き風デザイン)
* **レスポンシブ**: デバイス自動検出によるモバイル用UIの切り替え
